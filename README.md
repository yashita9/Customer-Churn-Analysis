# 📊 Customer Churn Analysis

## 🔍 Overview
This project analyzes customer data to understand why customers leave (churn) and identify key factors affecting retention.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Steps
- Data Cleaning (handled missing values, corrected data types)
- Exploratory Data Analysis (EDA)
- Data Visualization (count plots, histograms)

## 📌 Key Insights
- Customers with monthly contracts churn more
- Higher tenure reduces churn
- Higher monthly charges may increase churn risk

## 🧠 Concepts Covered
- EDA (Exploratory Data Analysis)
- Data Cleaning
- GroupBy operations
- Data Visualization

## 🚀 Conclusion
The project highlights important patterns in customer behavior and helps businesses understand how to improve customer retention.
